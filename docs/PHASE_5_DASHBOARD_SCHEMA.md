## 4. Dimension Name Mapping

The PE uses 5 quantum dimensions internally. The dashboard maps these to display names:

| Internal Name | Display Name | Color Hex | Description |
|---------------|-------------|-----------|-------------|
| `wit` | Warmth/Play | `#E86A5F` | Humor, playfulness, irreverence |
| `empathy` | Empathy | `#F5A623` | Emotional resonance, care |
| `precision` | Precision | `#4A90E2` | Data-driven, analytical |
| `patience` | Authority/Patience | `#7B4F9D` | Tolerance, measured responses |
| `entropy` | Stability/Entropy | `#50C878` | Unpredictability, creativity |

## 5. File Locations

| Artifact | Path Pattern |
|----------|-------------|
| Turn-level JSONL | `phase5_cycles/cycle_NNN/phase5_cycle_NNN_call_log.jsonl` |
| Call summaries | `phase5_cycles/cycle_NNN/phase5_cycle_NNN_summary.md` |
| Prosody heatmap | `phase5_cycles/cycle_NNN/phase5_cycle_NNN_prosody_heatmap.png` |
| Mode trajectory | `phase5_cycles/cycle_NNN/phase5_cycle_NNN_mode_trajectory.png` |
| State components | `phase5_cycles/cycle_NNN/phase5_cycle_NNN_state_components.png` |
| Anomalies | `phase5_cycles/cycle_NNN/phase5_cycle_NNN_anomalies.json` |

## 6. Acceptance Bands

| Metric | Acceptable Range | Alert If |
|--------|-----------------|----------|
| `speed_mod` | $[-0.08, +0.08]$ | Outside range |
| `silence_mod` | $[-3, +3]$ frames | Outside range |
| `quantum_entropy` | $[0.5, 1.4]$ | Outside range |
| `mood_score` | $[0.2, 0.9]$ | Sustained extremes |
| Turn duration | $[2s, 45s]$ | Outside range |
| Prosody bias active | `True` | Always false |
